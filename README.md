# Fuel-n-Fix

Fuel-n-Fix is a modern, user-friendly web application designed to help users easily locate nearby fuel stations and vehicle repair services. Leveraging Next.js, TypeScript, and Tailwind CSS, it delivers fast, responsive, and intuitive access to essential vehicle-related services.

## About

Fuel-n-Fix addresses the common challenge of finding nearby fuel stations and vehicle repair services quickly and reliably. It combines geolocation APIs with an interactive map interface, helping users navigate their surroundings to locate necessary services without hassle.

---

## Features

- **Geolocation-based search:** Automatically detect user location to find nearby services.
- **Search and filter:** Users can search for specific fuel stations or repair shops.
- **Interactive Map:** View results visually on a map with clickable markers.
- **Detailed Service Info:** View addresses, contact numbers, opening hours (if available).
- **Responsive Design:** Works seamlessly across devices, from mobile phones to desktops.
- **Fast Performance:** Built with Next.js for server-side rendering and optimized user experience.

## Installation

### Prerequisites

- Node.js (>= 14.x)
- Yarn package manager (preferred)

### Steps

```bash
# Clone the repository
git clone https://github.com/Chaitanyahoon/fuel-n-fix.git
cd fuel-n-fix

# Install dependencies
yarn install

# (Optional) Create a .env.local file if environment variables are needed
# For example:
# echo "NEXT_PUBLIC_MAP_API_KEY=your_api_key_here" > .env.local

# Start the development server
yarn dev
